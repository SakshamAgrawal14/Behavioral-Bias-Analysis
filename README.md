# Behavioral-Bias-Analysis
Analysis of cognitive biases using behavioral datasets and interactive experiments then bridging these ideas to real world economic impacts

## 📂 Project Structure
- data/ → datasets used for analysis  
- code/ → Python scripts for each bias  
- Presentation/ → slides and demo video  

## 🧠 Biases Covered
- Wason Selection Task (Confirmation Bias)
- Cognitive Reflection Test (CRT)
- Anchoring Effect
- Stroop Effect
